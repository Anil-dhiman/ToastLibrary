# ToastLibrary
A simple library to show toast
it works with all latest versions of android

 ## Steps 1
```bash
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  ```
  
  ## Steps 2
  
  ```bash
  dependencies {
	        implementation 'com.github.Anil-dhiman:ToastLibrary:1.0'
	}
  ```
